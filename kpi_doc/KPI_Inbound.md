## 1. Labour Efficiency
**Business Question:**</br>How efficiently is inbound/ Return labour converting hours into processed cartons?</br>  
**Formula:**
```
Labour Efficiency = Total Cartons Received / (Days Worked √ó Labour Count)
```   
**Importance:**</br>Core productivity indicator for staffing, scheduling, and labor optimization.</br>  
  

**Operational Meaning:**
- **High** = Strong staffing efficiency, optimized processes.  
- **Low** = Bottlenecks, idle time, under performance.  
 
**How to improve:**
- Training
- SOP refinement
- Workload balancing

## 2. Shift Productivity
**Business Question:**</br> How many inbound/ Return consignments are processed daily?</br>  
**Formula:**  
```
Productivity = Total Inbounds or Returns / Days Counted
```  

**Importance:**</br> Measures warehouse flow and receiving capacity.</br>  
**Operational Meaning:**
- High = good throughput
- Low = delays
  
**How to improve:**
- Slotting
- Shift alignment
- Elimination of delay points

## 3. Avg. Inbound/ Return Cycle Time
**Business Question:**</br> How long does it take to unload and put away items?</br>  
  
**Formula:**  
```
Avg. Cycle Time = (Putaway Completed time - Unloading Started time) / Days counted
```  
**Importance:**</br> Directly affects dock availability and how many trucks to accomodate within the operational time.</br>  

**Operational Meaning:**
- High = Slow inbound ops, congestion, poor coordination
- Low = Smooth flow, optimized handling

**How to improve:**
- Streamlined staging
- Better dock scheduling
- Trained teams

## 5. Incidents/ Accidents
**Business Question:**</br>Is the site operating safely?</br>  
**Formula:**  
```
Count of Incidents for the defined period + (Daily avg. with trend) 
```
 
 
**Importance:**</br>Core safety culture indicator. Fail here ‚Üí everything else collapses.  </br>  
  

**Operational Meaning:**
- **High** = Unsafe floor, poor compliance
- **Low** = Strong HSE culture

**How to improve:**
- Toolbox talks
- Hazard spotting
- Stricter SOP enforcement


## 6. Inbound Rejection % Over Time
**Business Question:**</br>Are suppliers sending compliant goods?</br>  
**Formula:**  
```
Rejection % = (Rejected Units / Total Inbounds) x 100
``` 
**Importance:**</br>Impacts inventory availability, failed-picks, and supplier reliability.  </br>  
  

**Operational Meaning:**
- **High** = Supplier quality issues
- **Low** = Stable upstream quality

**How to improve:**
- Supplier feedback
- Pre-alert checks
- QA standards

## 7. Top Suppliers by Inbound Value
**Business Question:**</br>Which suppliers drive the most inbound value?</br>  
 
**Importance:**</br>Helps identify critical suppliers and quality alignment needs.  </br>  
  
**Operational Meaning:**
- **High** = Strategic supplier, high inbound flow
- **Low** = Smaller contributor

**How to improve:**
- Partner alignment
- SLA tightening

## 8. Customer Returns Trend
**Business Question:**</br>Are customer returns increasing or stabilizing?</br>  

**Importance:**</br>Direct link to product quality, picking accuracy, and customer experience.  </br>  
  
**Operational Meaning:**
- **High** = Quality issues, picking errors, wrong dispatch
- **Low** = Healthy fulfillment quality

**How to improve:**
- Root cause checks
- QC at pick
- Packaging

## 9. Top Customers by Return on Sales Value (RSV%)
**Business Question:**</br>Which customers contribute most to return-related losses?</br>  
**Formula:**  
```
RSV % = (Returned Value / Sold Value) x 100
``` 
**Importance:**</br>Flags loss-driving accounts that need corrective action.  </br>  
  
**Operational Meaning:**
- **High** = High return behavior, service issues, poor product fit
- **Low** = Stable and profitable customer

**How to improve:**
- Account-level root cause analysis
- Delivery accuracy
- Education/ staff training
