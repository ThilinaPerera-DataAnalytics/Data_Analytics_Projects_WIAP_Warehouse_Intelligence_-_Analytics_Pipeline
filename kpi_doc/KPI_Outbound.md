## 1. Labour Efficiency
**Business Question:**</br>Is outbound manpower actually pulling its weight?  </br>  
**Formula:**  
```
Labour Efficiency = Total Picked Cartons / (Labour Count √ó Days)
```  
**Importance:**</br>Shows if outbound is staffed right or just burning hours.  </br>  
  
**Operational Meaning:**
- **High** = Strong flow, tight ops
- **Low** = Inefficiency, overstaffing, workflow gaps

**How to improve:**
- Better picking routes
- SOP clarity
- Zoning

## 2. Shift Productivity
**Business Question:**</br>How many orders can the warehouse push through daily?  </br>  
**Formula:**  
```
Orders per Day = Total Orders / Days Count
```   
**Importance:**</br>Reflects outbound throughput capacity.  </br>  
  
**Operational Meaning:**
- **High** = Good momentum
- **Low** = Bottlenecks in pick/pack/verify

**How to improve:**
- Slotting optimization
- Shift balancing

## 3. Order Fulfillment %
**Business Question:**</br>Are we delivering orders clean, accurate and drama-free?  </br>  
**Formula:**  
```
Order fulfillment % = (Loss-Free Qty / Total Ordered Qty) x 100
```  
**Importance:**</br>Percentage of quantity shipped without picking/packing errors. Direct hit to customer satisfaction and GP.  </br>  
  
**Operational Meaning:**
- **High** = Accurate picking
- **Low** = Mis-picks, training gaps, poor SOPs

**How to improve:**
- Improve stock management
- Pick validation/ checking

## 4. Avg. Picking Cycle Time
**Business Question:**</br>How long do pickers take from pick sheet issuance to finishing checking?  </br>  
**Formula:**  
```
Avg. Pick Time = (Pick Completed - Pick Sheet Issued) / Days Count
```   
**Importance:**</br>Shows picker efficiency and route planning quality.  </br>  
  

**Operational Meaning:**
- **High** = Slow picks, clutter, bad layout
- **Low** = Clean routes, good slotting

**How to improve:**
- Route optimization
- Better slotting
- Better GWP (Good Warehouse Practices)
- Improved rack allocation based on fast moving goods

## 5. Avg. Loading Cycle Time/ Stock dwelling time
**Business Question:**</br>Are checked orders stuck waiting too long before loading?  </br>  
**Formula:**  
```
Avg. PLoading Time = (Loading Completed - Pick Completed) / Days Count
```   
**Importance:**</br>Shows loader efficiency, cold chain maintenance effectiveness from dock-to-dispatch and route planning quality.  </br>  

**Operational Meaning:**
- **High** = Dock delays, improper cold chain maintenance 
- **Low** = Smooth dispatch flow

**How to improve:**
- Better staging layout
- Vehicle readiness
- GDP (Good Distribution Practices) training

## 6. Warehouse Throughput
**Business Question:**</br>Is outbound volume growing or sliding?  </br>  
**Formula:**
```
Total unit count over time
```   
**Importance:**</br>Reflects business demand + warehouse capacity resilience.  </br>  

**Operational Meaning:**
- **High** = Strong business flow
- **Low** = Seasonal drops, capacity constraints

**How to improve:**
- Better capacity planning
- Sales/ marketing/ merchandising improvements

## 7. Lost GP Due to Failed Picks

**Business Question:**</br>How much profit are we losing because of failed-picks?  </br>  
**Formula:**
```
Lost GP = Sum((Product Price - Product Cost) x Failed Qty)
```  
**Importance:**</br>Errors ‚Üí direct money loss.  </br>  
  

**Operational Meaning:**
- **High** = Stocks unavailability, sloppy picking, no proper checking mechanism
- **Low** = Stable accuracy

**Improvement:**
- Better stock management
- Supervise at aisle
- Scanning validations

## 8. Top Failed Pick Products (Pareto)

**Business Question:**</br>Which products cause the most picking pain?  </br>  
**Formula:**
```
Ranked by failed pick count
```   
**Importance:**</br>Shows where layout, stock availability sucks.  </br>  
  

**Operational Meaning:**
- **High** = Poor stock availability, higher products with issues count, labelling, slotting
- **Low** = Stable SKU handling

**How to improve:**
- SKU-level root cause
- Relabel/ Reslot

## 9. On-Time Dispatch (OTD %)

**Business Question:**</br>Are we dispatching orders when we promised?  </br>  
**Formula:**
```
OTD % = (Vehicles Dispatched Before Cutoff / Total Dispatches) √ó 100
```  
**Importance:**</br>Direct customer-experience KPI.  </br>  
  

**Operational Meaning:**
- **High** = Good fleet readiness + team discipline
- **Low** = Delays, vehicle issues, missing docs

**How to improve:**
- Scheduling
- Vehicle staging
- Improve picking and loading cycle times

## 10. Avg. Daily Vehicle Utilization %
**Business Question:**</br>Are we using the fleet properly or wasting runs?  </br>  
**Formula:**
```
Vehicle Utilization % = (Used Capacity / Total Capacity) x 100
```   
**Importance:**</br>Affects distribution cost per carton.  </br>  

**Operational Meaning:**
- **High** = Efficient planning
- **Low** = Underutilized fleet, cost leakage

**How to improve:**
- Route consolidation
- Load planning

## 11. Avg. Daily Revenue Trends
**Business Question:**</br>Is daily outbound revenue trending up or crashing?  </br>  
**Formula:**
```
Avg. Daily Revenue = (Ordered Qty √ó Product Price) / Days Count
```   
**Importance:**</br>Shows commercial strength & demand patterns.  </br>  
  
**Operational Meaning:**
- **High** = Strong sales volume
- **Low** = Demand drop or stock issues

**How to improve:**
- Stock availability
- Dispatch SLAs
- Better sales/ marketing stratergies
